# pallet-midnight-rpc

JSON-RPC interface for querying Midnight ledger state.

## Overview

This crate provides WebSocket/HTTP RPC methods for external clients to query contract state, ledger version, and [ZSwap](https://docs.midnight.network/learn/glossary#zswap) state root. It bridges the runtime API (`MidnightRuntimeApi`) to [JSON-RPC](https://docs.polkadot.com/polkadot-protocol/glossary/#json-rpc) endpoints accessible via the node's RPC server.

The RPC implementation uses jsonrpsee for async request handling and supports both HTTP and WebSocket transports. Each method internally calls the corresponding runtime API function, handling block hash resolution (defaulting to best block when not specified) and error translation. This layer is essential for DApp integration, allowing wallets and frontends to read contract state without direct chain access.

## API Specification

### RPC Methods

- **`midnight_contractState`** - Get contract state at address
- **`midnight_zswapStateRoot`** - Get ZSwap Merkle root
- **`midnight_apiVersions`** - Supported API versions
- **`midnight_ledgerVersion`** - Ledger version string

### Public Types

- [**`Midnight<C, Block>`**](https://github.com/midnightntwrk/midnight-node/blob/main/pallets/midnight/rpc/src/lib.rs#L50) - RPC server implementation
- [**`MidnightApi`**](https://github.com/midnightntwrk/midnight-node/blob/main/pallets/midnight/rpc/src/lib.rs#L22) - RPC trait (generated by jsonrpsee)
- [**`StateRpcError`**](https://github.com/midnightntwrk/midnight-node/blob/main/pallets/midnight/rpc/src/lib.rs#L60) - Contract/account state errors
- [**`BlockRpcError`**](https://github.com/midnightntwrk/midnight-node/blob/main/pallets/midnight/rpc/src/lib.rs#L70) - Block query errors

## Usage

```bash
# Get contract state
curl -X POST http://localhost:9944 \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"midnight_contractState","params":["<hex_address>"],"id":1}'

# Get ledger version
curl -X POST http://localhost:9944 \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"midnight_ledgerVersion","params":[],"id":1}'
```

## API Versioning

The `midnight_apiVersions` method returns supported API versions. Current versions:

- **Version 2** - Current API with `Result` return types

Version 1 (deprecated) used `Option` return types.
